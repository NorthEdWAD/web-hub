Here’s your **essentialist multi-day lesson plan** for teaching JavaScript IF statements, designed for four 90-minute classes:

---

### **Title**: JavaScript IF Statements: A Multi-Day Essentialist Lesson Plan
**Objective**: By the end of this lesson plan, students will deeply understand and confidently apply JavaScript IF statements in real-world scenarios.

---

### **Core Principles of Essentialism Applied**
1. **Focus on the vital few**: Only the most essential IF statement concepts and use cases are covered.
2. **Eliminate the trivial**: Avoids overwhelming details or edge cases that distract from core understanding.
3. **Encourage mastery**: Hands-on, repetitive practice with immediate feedback.

---

### **Materials Needed**
- Laptop/computer with a code editor (e.g., VS Code, Sublime Text)
- Browser for testing code (e.g., Chrome, Firefox)
- Projector or screen-sharing tool for demonstrations

---

### **Daily Breakdown**

#### **Day 1: Introduction to Conditional Logic**
- **Focus**: Understanding the purpose and basic syntax of IF statements.
- **Activities**:
  1. **Concept Introduction** (30 min): Discuss the role of conditional logic in programming. Introduce the syntax of IF statements with simple examples.
  2. **Live Coding Demo** (25 min): Write and explain 3-4 basic IF statement examples (e.g., checking if a number is positive).
  3. **Guided Practice** (25 min): Students write their first IF statements with instructor guidance (e.g., checking if a user is logged in).
  4. **Q&A/Reflection** (10 min): Address questions and reflect on the importance of conditional logic.

#### **Day 2: IF-ELSE and ELSE-IF Statements**
- **Focus**: Expanding conditional logic with ELSE and ELSE-IF.
- **Activities**:
  1. **Review and Warm-Up** (10 min): Quick recap of Day 1 with a 5-minute quiz on basic IF statements.
  2. **Introducing ELSE and ELSE-IF** (25 min): Explain the syntax and purpose of ELSE and ELSE-IF with clear examples (e.g., grading systems).
  3. **Pair Programming** (40 min): Students work in pairs to solve 3-4 problems using IF-ELSE and ELSE-IF statements.
  4. **Wrap-Up Discussion** (15 min): Discuss common pitfalls and best practices for writing clear conditional logic.

#### **Day 3: Nested IF Statements and Logical Operators**
- **Focus**: Combining conditions with logical operators and nested IF statements.
- **Activities**:
  1. **Review and Challenge** (15 min): Solve a complex IF-ELSE problem as a class to review Day 2.
  2. **Logical Operators (&&, ||, !)** (25 min): Introduce logical operators and demonstrate their use in IF statements (e.g., validating user input).
  3. **Nested IF Statements** (20 min): Explain and demo nested IF statements with a focus on readability and simplicity.
  4. **Hands-On Lab** (30 min): Students build a mini-project (e.g., a simple quiz game) using nested IFs and logical operators.

#### **Day 4: Real-World Applications and Debugging**
- **Focus**: Applying IF statements to real-world scenarios and debugging common errors.
- **Activities**:
  1. **Real-World Code Review** (20 min): Analyze and discuss IF statements in open-source projects or real-world code snippets.
  2. **Debugging Workshop** (30 min): Introduce common errors (e.g., syntax mistakes, logical flaws) and practice debugging.
  3. **Capstone Activity** (30 min): Students create a small program (e.g., a discount calculator) using all concepts learned.
  4. **Reflection and Next Steps** (10 min): Reflect on the week’s learning and discuss how to apply these skills to future projects.
