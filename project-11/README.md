# 🎨 **Interactive Project**  

**Objective:** 

- *Create a **responsive**, two-page website on a classroom-friendly topic that:*
  - uses CSS Grid for the general page layout
  - uses JavaScript to make the site interactive
    - Uses JS functions to change the page styling *on the fly/dynamically*
    - Contains a JavaScript-powered slideshow or animation about the site's topic/theme

## ✅ **Requirements**

Your project is complete when you have:

- [ ] Created and linked the two pages to one another
- [ ] Used CSS Grid to make your pages responsive
- [ ] Used JavaScript functions to style page content *dynamically/on the fly*
- [ ] Used JavaScript to add a simple animation/slideshow to one of your pages
- [ ] Tested and debugged your code
- [ ] Uploaded your code and image files to GitHub
- [ ] Shared the link to your GitHub repo (using the GitHub URL form)
