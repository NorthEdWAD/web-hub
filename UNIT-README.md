# Unit 01: HTML & CSS Basics

## Overview
By the end of the unit, students will be able to create and style simple web pages using semantic HTML and basic CSS.


## How to Use This Unit  

- **Always** do your best work
- Use the background info and examples provided
- Apply what you've learned to your projects
- Debug your work **as you complete each part of your project** 

---

## Learning Objectives
By completing this unit, students will:
- Understand the structure of an HTML document
- Use semantic HTML elements to create accessible web pages
- Apply basic CSS to style text, colors, and layouts
- Build a simple multi-page website
