### Hints

- Set `width` of grid container to 750px
- Set `height` of grid container to 600px
- Use CSS `margin: 0 auto;` to center grid container horizontally in browser window
- Use **grid template areas** to define a grid with:
  - 3 columns
  - 4 rows
- Use CSS to apply `10px` of padding to all your grid areas
- Give each grid area its own background color
- Your `grid-container` will have six (6) grid areas
  - What HTML5 elements could you use to create grid areas named `header` and `footer`?
