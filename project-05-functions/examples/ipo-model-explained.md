### The IPO Model in Programming

- How to organize your code in your script so you get the correct results

#### I = INPUT

- Data JavaScript needs to do its job or to perform a calculation
  - Positioning alien spaceship in a game
  - Calculating average quiz score
 
#### p = PROCESS

- Do something with the input, change it/manipulate it
  - What about calculating an AVERAGE of two quiz scores?
 
#### O = OUTPUT

- Output = results or answer
   - Where do I want to show the output?
   - How do I want to format or style the output?
